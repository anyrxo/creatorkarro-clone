================================================================================
TROUBLESHOOTING QUICK FIXES
================================================================================

Site Down:
  1. Check: pm2 status
  2. View logs: pm2 logs --err
  3. Restart: pm2 restart all
  4. If fails: Rollback deployment
  5. Contact hosting support

Analytics Not Tracking:
  1. Check browser console (F12)
  2. Verify: NEXT_PUBLIC_GA_MEASUREMENT_ID set
  3. Test in incognito (disable ad blockers)
  4. Check GA4 DebugView (add ?debug_mode=true)
  5. Wait 24-48 hours for reports

API Not Responding:
  1. Check: pm2 logs iimagined-api
  2. Test: curl http://localhost:5001/api/health
  3. Restart: pm2 restart iimagined-api
  4. Check rate limiting
  5. Verify environment variables

Build Failures:
  1. Clear cache: rm -rf .next node_modules/.cache
  2. Check: npx tsc --noEmit
  3. Reinstall: rm -rf node_modules && npm install
  4. Check: npm run lint
  5. Review recent code changes

SEO Health Score Dropped:
  1. Run: npm run seo:health
  2. Check: Recent code changes
  3. Verify: Schema markup intact
  4. Check: Titles not removed
  5. Run: npm run seo:guard (pre-commit hook)

Performance Issues:
  1. Check: npx lighthouse https://yourdomain.com
  2. Verify: Images optimized
  3. Check: CDN configuration
  4. Test: Database query performance
  5. Review: Server resources (CPU, RAM)
